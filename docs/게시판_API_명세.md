# 게시판 API 명세 안내

이 파일은 초기 설계 단계에서 작성된 초안 파일명과의 링크 호환을 위해 남겨 둔다.

현재 백엔드 구현과 일치하는 게시판 API 명세는 [`REST_API_명세_게시판_최종.md`](./REST_API_명세_게시판_최종.md)를 기준으로 한다. 다음 항목을 포함한 기존 초안의 내용은 더 이상 유효하지 않다.

- 작성자 입력·응답 필드
- `search_type` Query와 `sort=best`
- `{ "verified": true }` 형태의 비밀번호 검증 응답
- 추천 수만 반환하는 별도 추천 응답 스키마
- 비밀번호 평문 저장 또는 별도 `LikeResponse` 클래스

여행 일정의 필드별 상세 합의와 프런트 전환 사항은 아래 문서를 함께 참고한다.
